YouTube Live Chat Websocket API

Programmatic reading of youtube live chat made trivial


Example Usage

const ws = new WebSocket('wss://ytc.taya.one/@ChannelHandle')

ws.onmessage = e => {
    const data = Json.parse(e.data)
    if(data.type == 'message') {
        console.log(data.author.name, data.text)
    }
}
